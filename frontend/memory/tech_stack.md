## 核心技术栈

- **框架**: React 19.1.0 + UmiJS Max 4.3.24
- **UI**: Ant Design 5.25.4 + Pro Components 2.7.19
- **语言**: TypeScript 5.6.3 (严格模式)
- **路由**: React Router DOM 7.13.1
- **状态管理**: Umi Max 内置 model + initialState
- **请求**: Umi request (基于 Axios 封装)
- **图表**: ECharts 6.0.0
- **拖拽**: react-dnd + react-dnd-html5-backend
- **日期**: Dayjs (moment2dayjs 插件迁移)
- **构建**: Webpack (Umi 内置)
- **代码规范**: Biome 2.0.6
- **测试**: Jest + Testing Library

## 项目定位

高校智能排课系统，包含基础数据管理、规则配置、智能排课、课表预览与统计、拖拽编辑等功能模块。

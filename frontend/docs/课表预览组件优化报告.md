## 问题描述

### 原始问题

周视图使用 CSS Grid 布局渲染课表时，跨节课程通过 `gridRow: span N` 跨越多行，但被占据的后续单元格仍然渲染空白 div，导致：

1. **Grid 实际渲染行数** = `TIME_SLOTS.length + 跨节额外行数`
2. **表格底部出现多余空白行**
3. **布局高度超出预期**，需要 `overflow: hidden` 强制裁剪

### 问题示意图

```

│ 第 2 节 │ [被占据 - 空白]  │ ... │  ← 多余行 │
```

## 解决方案

### 核心策略

参考 `ScheduleTable` 组件实现，采用**固定行数 + 明确 gridRow** 的策略：

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| 行数控制 | `gridAutoRows` | `gridTemplateRows` |
| 高度限制 | `maxHeight + overflow` | 固定行数 × 行高 |
| 单元格定位 | 隐式布局 | 显式 `gridRow` |

## 技术总结

### CSS Grid 布局要点

1. **`gridTemplateRows` vs `gridAutoRows`**
   - `gridTemplateRows`: 明确定义每行高度和数量
   - `gridAutoRows`: 自动创建新行，可能导致溢出

2. **`gridRow` 语法**
   - `${start} / ${end}`: 从第 start 行到第 end 行
   - `${start} / span ${count}`: 从第 start 行跨越 count 行

3. **单元格定位策略**
   - 每个单元格显式指定 `gridRow` 和 `gridColumn`
   - 被跨节占据的单元格只占单行，不跨行
   - 跨节课程仅在起始单元格渲染卡片

### 最佳实践

1. **固定行数**: 使用 `gridTemplateRows` 明确定义行数
2. **显式定位**: 每个单元格指定 `gridRow` 和 `gridColumn`
3. **分离关注点**: 布局逻辑与渲染逻辑分离
4. **类型安全**: TypeScript 确保数据结构正确

## 导出功能说明

### 功能概述

支持将当前课表导出为 **PNG 图片** 或 **PDF 文档**，方便打印和分享。

### 技术实现

```jsx
// 依赖
import html2canvas from 'html2canvas';
import jsPDF from 'jspdf';
```

### 导出为图片

```jsx
const exportToImage = async () => {
  const canvas = await html2canvas(scheduleContentRef.current, {
    scale: 2,              // 2 倍缩放，提高清晰度
    backgroundColor: '#ffffff',
    useCORS: true,
    logging: false,
  });

  canvas.toBlob((blob) => {
    const url = URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.href = url;
    link.download = `课表 - 第${currentWeek}周.png`;
    link.click();
  }, 'image/png');
};
```

### 导出为 PDF

```jsx
const exportToPDF = async () => {
  const canvas = await html2canvas(scheduleContentRef.current, {...});
  const imgData = canvas.toDataURL('image/png');

  const pdf = new jsPDF({
    orientation: 'landscape',  // 横向
    unit: 'mm',
    format: 'a4',
  });

  pdf.addImage(imgData, 'PNG', 0, 0, imgWidth, imgHeight);
  pdf.save(`课表 - 第${currentWeek}周.pdf`);
};
```

### 使用方式

1. 点击"导出课表"按钮
2. 选择"导出为图片"或"导出为 PDF"
3. 文件自动下载到本地

### 注意事项

- 导出时会捕获当前视图（周视图/日视图/列表视图）
- 使用 `ref` 引用课表内容区域
- 导出过程中按钮显示 loading 状态
